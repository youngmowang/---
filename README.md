# AutoWord - 智能背单词助手

AutoWord 是一款基于 C++ 和 Qt 6 开发的现代化背单词软件。它不仅仅是一个简单的词汇列表，更内置了先进的 **FSRS (Free Spaced Repetition Scheduler)** 记忆算法，就像一位智能私教，根据你的记忆曲线自动安排复习计划，帮你用最少的时间记住最多的单词。

---

## 📖 如何使用 (用户指南)

如果你是第一次使用本软件，请按照以下步骤操作：

### 1. 准备词库
软件运行需要数据支持。请准备一个 CSV 格式的单词表文件（UTF-8 编码）。
**格式要求**：`单词,音标,释义` 或 `单词,释义`
> **示例文件 (test.csv):**
> ```csv
> apple,[ˈæpl],苹果
> banana,[bəˈnɑːnə],香蕉
> computer,,计算机
> ```

### 2. 导入单词
1.  启动软件，点击左侧导航栏底部的 **设置 (Settings)** 按钮。
2.  找到 **"词典管理"** 区域。
3.  点击 **"导入词典"** 按钮，选择你准备好的 CSV 文件。
4.  导入成功后，系统会自动创建一个新的单词书。

### 3. 开始学习 (Study)
这是核心功能。系统会根据算法给你推送卡片。
1.  点击左侧 **学习 (Study)** 按钮。
2.  在顶部下拉框选择你要学习的 **单词书**。
3.  **操作流程**:
    *   屏幕显示单词（和发音）。
    *   **思考**它的含义。
    *   按 **`空格键`** (或点击 "显示答案") 查看正确释义。
    *   **自我评分** (决定下一次复习时间):
        *   按 **`1`** (重来): 完全没记住，马上重来。
        *   按 **`2`** (困难): 勉强想起来，间隔短一点。
        *   按 **`3`** (良好): 记得不错，正常间隔。
        *   按 **`4`** (容易): 这词太简单了，很久以后再见。

### 4. 单词测试 (Test)
检测学习成果，支持两种模式。
1.  点击左侧 **测试 (Test)** 按钮。
2.  **拼写模式**: 听发音或看释义，键盘输入单词拼写（答错会有蜂鸣音提示）。
3.  **回忆模式**: 经典的四选一选择题，快速过词。

### 5. 查看进度 (Dashboard)
点击左侧 **仪表盘 (Dashboard)** 回到首页。
*   **数据卡片**: 查看总单词量、已熟记单词数、今日待复习数。
*   **趋势图表**: 这里的折线图记录了你 **最近7天** 的复习单词数量，看着曲线主要上升，满满的成就感！

## 功能特性

*   **科学记忆**: FSRS 算法让复习更高效。
*   **数据可视化**: 首页直观展示学习趋势。
*   **单词搜索**: 在预览页可以实时搜索单词。
*   **语音朗读**: 内置 TTS 引擎，单词会发音。
*   **夜间模式**: 可以在设置里切换深色/浅色主题。

## 编译与构建 (开发人员指南)

如果您需要从源码编译本项目（例如进行课程设计验收），请参考以下步骤。

### 开发环境要求
*   **编译器**: C++17 (MinGW 64-bit 推荐)
*   **框架**: Qt 6.x (包含 Widgets, Network, Sql, Multimedia, TextToSpeech)
*   **构建工具**: CMake 3.16+

### 编译步骤 (Windows PowerShell 示例)

假设你的项目在 `f:\autoword`，CMake 安装在 `E:\Qt\Tools\CMake_64\bin`。

1.  **进入项目目录**:
    ```powershell
    cd f:\autoword
    ```

2.  **创建并进入构建目录**:
    ```powershell
    mkdir build
    cd build
    ```

3.  **配置并编译**:
    ```powershell
    # 使用指定路径的 CMake 进行配置
    & "E:\Qt\Tools\CMake_64\bin\cmake.exe" ..
    
    # 开始编译
    & "E:\Qt\Tools\CMake_64\bin\cmake.exe" --build .
    ```

4.  **运行程序**:
    ```powershell
    .\AutoWord_v1.8.exe
    ```
